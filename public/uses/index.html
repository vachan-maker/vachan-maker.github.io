Uses

Laptop

I use a Dell Inspiron 5447, it runs on Debian 12 (Bookworm). I think it is about 10 years old and it works alright. But it is not in the best condition so I can’t really carry it anywhere.

Software

  • Mozilla Firefox is my default web browser
  • I use Inkscape for designing some simple posters.
  • LibreOffice for editing documents
  • I use GoVarnam to type Malayalam
  • Kdenlive for video editing
  • GIMP for image editing
  • Audacity or Tenacity for audio editing. I have both of them installed.
  • I use Gedit primarily for text editing. Sometimes I use Visual Studio. I am also learning to use Neovim.
  • For recording my screen, I use OBS.

Phone

I use a Samsung Galaxy M13 running Android 13

Software

  • Pretty much all the google apps and samsung apps that came preinstalled
  • Tusky is my Mastodon client
  • Feeder for RSS
  • Firefox Nightly is my default browser
  • PanoScrobbler for tracking music I listen to
  • GitHub
  • Next Player and VLC for watching videos
  • Aegis for storing TOTP codes
  • Aniyomi for watching anime
  • Seal for downloading videos from YouTube and other sites
  • KDE Connect for connecting my phone to my laptop
